## 📊 基础配置参数（所有模板通用）

### 成本配置
```
汇率设置：1 CNY = 5 THB
货代成本：2元 = 10 THB（腾飞速达）
包装成本：1元 = 5 THB
基础固定成本：3元 = 15 THB
```

### 平台费用
```
平台佣金率：5%
平台手续费：1%
售后成本率：10%
```

### 计算公式
```
售价 = (产品成本 + 固定成本) / (1 - 佣金率 - 手续费 - 售后率 - 目标利润率)
```

## 🎯 模板A：低价引流款

### 模板参数

```
适用市场：TikTok Shop Thailand
目标利润率：25%
```

### 成本结构
```
产品成本：15元 = 75 THB
货代成本：2元 = 10 THB
包装成本：1元 = 5 THB
```

### 计算验证
```

售价 = 90 / 0.59 = 152.5 THB ≈ 155 THB（四舍五入）
```

### 售价范围

```
基础售价：155 THB
建议定价：150-180 THB
预期利润：37-45 THB（25-30%）
```

## 📝 在妙手ERP中的操作步骤

### Step 1：创建定价模板
```
1. 登录妙手ERP
2. 进入：商品管理 → 定价模板 → 新建模板
3. 根据上述3个模板的参数，依次创建
```

### Step 2：填写模板信息（以模板B为例）
```
模板名称：泰国站-标准平衡-35%利润
适用市场：TikTok Shop Thailand
汇率配置：1 CNY = 5 THB

成本参数：
- 产品成本：18元（可变）
- 货代成本：2元
- 包装成本：1元
- 固定成本合计：21元

费用参数：
- 平台佣金率：5%
- 平台手续费：1%
- 售后成本率：10%

目标利润率：35%

计算结果：
- 售价：215 THB
- 预期利润：75 THB
```

### Step 3：套用模板到商品
```
1. 进入商品编辑/采集页面
2. 勾选需要定价的商品
3. 点击"更多操作" → "套用定价模板"
4. 勾选"目标站点"：✓ TikTok Shop Thailand
5. 选择目标店铺：Nestify
6. 选择定价模板：泰国站-标准平衡-35%利润
7. 点击"确认套用"
```

### Step 4：验证价格
```
1. 查看商品详情页
2. 查看"本地展示价"字段
3. 应显示：约215 THB
4. 忽略"税前售价"字段
```

### Step 5：上架验证
```
1. 发布商品到TikTok Shop
2. 进入TikTok后台 → 管理商品
3. 查看"零售价"字段
4. 应显示：约215 THB
```

## ⚠️ 常见问题排查

### 问题1：价格不对
```
检查项：
✓ 定价模板是否已创建
✓ 模板参数是否正确填写
✓ 是否勾选了"套用定价模板"
✓ 是否勾选了"目标站点"
✓ 汇率是否正确（1:5）
```

### 问题2：利润率计算错误
```
检查项：
✓ 产品成本是否填写
✓ 货代成本是否填写（2元）
✓ 固定成本是否正确
✓ 费用率是否正确
✓ 目标利润率是否设置
```

### 问题3：上架后价格不同
```
检查项：
✓ 妙手价格：查看"本地展示价"
✓ TikTok价格：查看"管理商品-零售价"
✓ 汇率是否需要更新
✓ 是否开启了平台自动定价
```

### 问题4：多个SKU价格不一致
```
检查项：
✓ 每个SKU的成本是否一致
✓ 是否所有SKU都套用了模板
✓ 查看最小SKU的价格
✓ 必要时单独调整异常SKU
```

## 💡 高级技巧

### 技巧1：批量调整价格
```
场景：汇率变动需要统一调价

操作：
1. 更新定价模板中的汇率
2. 批量选择商品
3. 重新套用定价模板
4. 系统自动更新所有商品价格
```

### 技巧2：促销活动临时调价
```
场景：大促期间需要临时降价

操作：
1. 创建"大促模板"（利润率20%）
2. 活动前批量套用
3. 活动结束后恢复原模板
4. 保留模板方便下次使用
```

### 技巧3：分品类定价
```
场景：不同品类利润率不同

操作：
宠物用品 → 模板B（35%）
厨房用品 → 模板A（25%）
收纳用品 → 模板C（45%）

按品类分别套用对应模板
```
