## 🔄 自动化流程设计

### 程序性记忆自动化逻辑
```python
# 伪代码：自动提取和记录机制
def process_task_result(result):
    """所有任务完成后自动调用"""
    
    if result.success:
        # 成功任务处理
        extract_key_steps(result.execution_log)
        identify_patterns(result.success_factors)
        update_skill_md(result.key_learnings)
        add_to_best_practices(result.replicable_insights)
        
        # 重要事件标记
        if result.impact > 50:  # 高影响事件
            mark_permanent(result)
            add_to_fossil_memory(result)
            
    else:
        # 失败任务处理
        analyze_failure(result.error_trace)
        identify_root_cause(result.execution_context)
        create_failure_pattern(result.failure_signature)
        
        # 生成解决方案
        generate_solution(result.error_type)
        add_to_failure_patterns(result.lesson_learned)
        
        # 更新检查清单
        update_prevention_checklist(result.preventive_measures)
```

### 常见模式自动识别
**选品工作中的模式识别**:

#### 高价值商品模式
**特征**:
- 价格区间: ฿50-200
- 3日销量: >50个
- 包邮/本地仓: true
- 功能展示简单: true

**应用**: 自动提升优先级，优先推荐

#### 风险商品模式
**特征**:
- 价格过高: >฿300
- 体积过大: >30cm
- 重量过重: >500g
- 安装复杂: true

**应用**: 自动标记为低优先级或直接过滤

#### 潜力新品模式
**特征**:
- 3-7天新品
- 销量快速增长
- 评论积极
- 达人合作增加

**应用**: 重点跟踪和观察
