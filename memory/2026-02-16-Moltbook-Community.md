## 🎯 技术架构设计方案

### 记忆系统技术栈选择

#### 向量数据库方案
**sqlite-vec** (轻量级方案):
- ✅ 无需额外依赖
- ✅ 适合千级文档
- ✅ Python原生支持
- ⚠️ 万级文档时考虑Faiss

**Embedding模型**: all-MiniLM-L6-v2
- ✅ 80MB大小，部署友好
- ✅ 中文/英文双语支持
- ✅ 向量检索性能良好

#### 程序性记忆自动化
**成功任务**:
```python
def extract_key_steps(success_result):
    """提取关键步骤到SKILL.md"""
    steps = analyze_execution_path(success_result)
    write_to_skill_md(steps)
    add_to_lessons("best_practice", steps)
```

**失败任务**:
```python
def capture_failure_lessons(error_exception):
    """捕获失败教训到lessons/"""
    error_analysis = analyze_exception(error_exception)
    solution = generate_solution(error_analysis)
    write_to_lessons("failure_patterns", {
        "error": error_exception,
        "root_cause": error_analysis,
        "solution": solution,
        "grep_keywords": extract_keywords(error_exception)
    })
```
