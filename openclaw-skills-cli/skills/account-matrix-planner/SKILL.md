# 账号矩阵规划技能

智能规划TikTok Shop账号矩阵，最大化账号起量效果，降低封号风险。

## 信息

- **Version**: 1.0.0
- **Author**: OpenClaw Team
- **Tags**: account-matrix, tiktok, growth, risk-management
- **Icon** 📱

## 功能

- 📊 账号数量规划（主号+副号+备用号）
- 🌐 IP分配和隔离策略
- 🎯 内容差异化方案
- ⚖️ 风险控制机制
- 📈 起量策略优化

## 使用方法

```bash
openclaw run account-matrix-planner --target-sales="100000" --budget="5000" --risk="medium"

openclaw run account-matrix-planner --target-sales="50000" --accounts="5" --market="东南亚"
```

## 参数说明

- `--target-sales`: 目标月销金额（$50000、$100000等）
- `--budget`: 每月预算（$5000、$10000等）
- `--risk`: 风险承受度（low/medium/high）
- `--accounts`: 指定账号数量（可选，会自动计算推荐值）
- `--market`: 目标市场（美区/欧盟/日本/东南亚）

## IP隔离策略

### Level 1：高隔离（主账号）
- **IP类型**：独立专用IP
- **来源**：优质代理服务商
- **稳定性**：99.9%
- **成本**：$200-500/月
- **风险**：最低

### Level 2：中隔离（副账号）
- **IP类型**：住宅IP池
- **来源**：多区域住宅IP
- **稳定性**：95%
- **成本**：$50-150/月
- **风险**：中等

### Level 3：低隔离（备用账号）
- **IP类型**：移动IP/数据中心IP
- **来源**：动态IP
- **稳定性**：85%
- **成本**：$20-50/月
- **风险**：较高

**关键原则**：
- ✅ 禁止同一IP登录多个账号
- ✅ 定期更换IP（每周1次）
- ✅ 建立IP池，避免单点过载
- ✅ 监控IP健康度，及时封禁异常

## 封号风险评估

### 🔴 高风险行为（绝对禁止）
- 同IP登录多个账号
- 100%复制相同内容到多账号
- 同时发布完全相同内容
- 跨账号互动（点赞/评论/分享）
- 短时间内大规模加粉

### 🟡 中风险行为（谨慎操作）
- 多账号使用相似内容
- 发布频率不稳定
- 频繁更换设备
- 大量私信营销
- 过度优惠广告

### 🟢 低风险行为（推荐）
- 账号内容差异化明显
- 稳定的发布频率
- 定期更换IP
- 自然增长粉丝
- 高质量互动

## 应急预案

### 封号应对流程

1. **立即响应**（0-1小时）
   - 备用账号立即上线
   - 数据库立即备份
   - 暂停所有其他账号活动

2. **分析原因**（1-24小时）
   - 检查封号原因
   - 评估对其他账号的影响
   - 制定恢复计划

3. **恢复运营**（1-3天）
   - 新账号启动（如果主账号无法恢复）
   - 内容投放调整
   - 预算重新分配

4. **长期预防**（持续）
   - 更新风控策略
   - 增加账号数量
   - 优化内容差异化
