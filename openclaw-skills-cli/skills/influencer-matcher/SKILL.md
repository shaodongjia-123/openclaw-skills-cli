# 达人匹配推荐技能

智能匹配TikTok Shop适合的达人，预估合作效果，优化达人带货策略。

## 信息

- **Version**: 1.0.0
- **Author**: OpenClaw Team
- **Tags**: influencer, tiktok,达人, partnership, earnings-forecast
- **Icon**: 🤝

## 功能

- 🎯 达人精准匹配（基于粉丝画像和产品匹配度）
- 📊 达人KPI分析（粉丝数、互动率、转化率）
- 💰 合作模式推荐（纯佣/提成/固定费用）
- 📈 收益预测（ROI预估）
- ⚠️ 风险评估（历史数据/口碑）
- 📋 合同要点提醒

## 使用方法

```bash
# 根据产品匹配达人
openclaw run influencer-matcher --product="厨房小家电" --market="东南亚" --budget="2000"

# 根据预算推荐达人
openclaw run influencer-matcher --budget="5000" --followers="1000-10000" --market="泰国"

# 达人ROI预测
openclaw run influencer-matcher --influencer-id="@达人账号" --cost="500" --product-price="199"
```

## 参数说明

- `--product`: 产品描述或品类
- `--market`: 目标市场（美区/欧盟/日本/东南亚/泰国/越南/马来西亚）
- `--budget`: 合作预算（$500、$1000、$2000、$5000）
- `--followers`: 达人粉丝数范围（1000-10000、10000-50000等）
- `--influencer-id`: 特定达人账号
- `--cost`: 达人合作费用
- `--product-price`: 产品售价
- `--mode`: 匹配模式（精准/快速/广泛）

## 达人类型分类

### KOL（关键意见领袖）
- 粉丝：100,000+
- 互动率：15-30%
- 成本：$1,000-10,000
- 适用：品牌曝光、新品发布

### KOC（关键意见消费者）
- 粉丝：10,000-100,000
- 互动率：20-40%
- 成本：$500-2000
- 适用：日常推广、产品测评（推荐）

### 微达人
- 粉丝：1,000-10,000
- 互动率：30-50%
- 成本：$200-500
- 适用：长尾产品、广撒网

## 合作模式类型

### 纯佣模式（最常用）
- **适用**：高转化产品、小型KOL
- **优点**：无风险，激励强
- **缺点**：达人不一定高质量
- **佣金比例**：15-25%

### 固定费用模式
- **适用**：KOL、大V、新品发布
- **优点**：确保质量
- **缺点**：风险高
- **费用**：$500-10,000

### 混合模式（最佳）✨
- **适用**：中等KOL、稳定合作
- **优点**：平衡质量和激励
- **预算分配**：固定30%-50%，佣金50%-70%

### 阶梯佣金模式
- **适用**：高潜力产品
- **阶梯**：
  - <100单：15%
  - 100-500单：20%
  - >500单：25%

## 风险评估体系

### 🔴 高风险
- 粉丝数据异常增长
- 互动率过高（>50%）
- 历史合作评价低
- 频繁更换合作产品
- 评论区负面占比>30%

### 🟡 中风险
- 粉丝增长稳定但缓慢
- 互动率适中（10-25%）
- 历史合作评价中等
- 产品类型较复杂

### 🟢 低风险
- 粉丝增长健康（每月+5-15%）
- 互动率合理（20-35%）
- 历史合作评价良好
- 合作稳定

## 合同要点检查清单

### 商务条款

- ☑️ 内容质量要求
- ☑️ 品牌露出要求
- ☑️ 产品展示位置

- ☑️ 归属权（视频所有权）
- ☑️ 竞品限制（是否允许其他同类品牌）
- ☑️ 争议解决方式

### 技术条款

## ROI预测模型

### 基础模型
```
ROI = （销量 × 客单价 × 0.8 - 合作总成本）/ 合作总成本
```

### 影响因子
| 因素 | 权重 | 影响 |
|------|------|------|
| 达人精准度 | 30% | ±20% ROI |
| 互动率高低 | 25% | ±15% ROI |
| 产品价格匹配 | 20% | ±10% ROI |
| 发布时间 | 15% | ±8% ROI |
| 产品质量 | 10% | ±5% ROI |
