# 社交媒体趋势监控技能

监控东南亚地区TikTok、Facebook等社交媒体平台的热门话题和趋势。

## 信息

- **Version**: 1.0.0
- **Author**: OpenClaw Team
- **Tags**: social, monitoring, trending, analysis
- **Icon**: 📊

## 功能

- 📈 实时监控热门话题和Hashtag
- 🔍 发现潜在的热门产品趋势
- 💡 竞品选品数据参考
- 🌍 针对东南亚地区（泰国、越南、马来西亚等）
- 📊 趋势数据对比分析（日/周/月）

## 使用方法

```bash
# 监控TikTok泰国热门话题
openclaw run social-trend-monitor --platform="TikTok" --region="泰国" --top=10

# 监控特定Hashtag趋势
openclaw run social-trend-monitor --hashtag="#好物推荐" --platform="TikTok"`

# 监控多平台趋势对比
openclaw run social-trend-monitor --platform="all" --top=5
```

## 参数说明

- `--platform`: 平台（TikTok/抖音/Facebook/Instagram/YouTube）
- `--region`: 地区（泰国/越南/马来西亚/新加坡/菲律宾/印度尼西亚）
- `--top`: 显示前N个热门话题
- `--hashtag`: 特定Hashtag监控
- `--days`: 天数范围（1/7/30天，默认7天）

## 东南亚地区特色

### 泰国 🇹🇹
- 热门：美食、旅游好物、时尚
- 文化：注重真实体验、用户评价
- 语言：泰语 + 英语
- 流量高峰：18:00-23:00

### 越南 🇻🇳
- 热门：美妆、搞笑、音乐、时尚
- 文化：年轻化、娱乐性强
- 语言：越南语
- 流量高峰：19:00-23:00

### 马来西亚马来 🇲🇲
- 热门：母婴、数码产品、家居
- 文化：价格敏感，重视性价比
- 语言：马来语 + 英语
- 流量全年较为均匀

### 新加坡 🇸🇬
- 热门：美食、生活好物、时尚、数码
- 文化：高质量、精致生活
- 语言：英语
- 流量高峰：12:00-14:00，19:00-23:00

## 趋势类型分析

### 上升趋势（⬆️）
- 新兴类目，用户增长快
- 热门话题快速攀升
- 相关视频近期增加

### 稳定趋势（➡️）
- 长期热门话题
- 稳定用户群体
- 销售转化稳定

### 下降趋势（⬇️）
- 季节性话题
- 热度逐渐消退
- 市场饱和迹象

## 选品策略建议

### 上升趋势区域
- 抓早期红利，竞争对手少
- 研发成本低
- 用户增长空间大

### 稳定趋势区域
- 热门但竞争激烈
- 需要差异化定位
- 重视产品质量和评价

### 趋势下降区域
- 避免市场饱和类目
- 或有价格优势、品牌优势再考虑
