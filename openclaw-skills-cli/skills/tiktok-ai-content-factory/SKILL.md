# TikTok AI内容工厂技能（增强版）

一站式AI内容生产平台，为TikTok Shop账号矩阵批量生成高质量、差异化的短视频内容。

## 信息

- **Version**: 2.0.0（增强版）
- **Author**: OpenClaw Team
- **Tags**: tiktok, content-factory, ai, batch-production, automation
- **Icon**: 🏭

## 功能

- 🎬 批量视频脚本生成（30秒/60秒/3分钟）
- ✍️ AI标题生成（3秒钩子必备）
- 🏷️ 热门Hashtag推荐（热门+长尾）
- 🎵 BGM音乐推荐（卡点音乐）
- 🔤 多语言文案支持（中文/英文/泰语/越南语/马来语）
- 📊 A/B测试内容变体生成
- 🎭 账号差异化内容输出

## 使用方法

```bash
# 单条内容生成
openclaw run tiktok-ai-content-factory --product="厨房小家电" --style="幽默" --duration="30秒"

# 批量生成（为账号矩阵）
openclaw run tiktok-ai-content-factory --product="厨房小家电" --accounts="5" --batch

# A/B测试生成
openclaw run tiktok-ai-content-factory --product="厨房小家电" --variants="3" --market="东南亚"
```

## 参数说明

- `--product`: 产品或服务描述
- `--style`: 内容风格（幽默/专业/故事/教育/推广）
- `--duration`: 视频时长（15秒/30秒/60秒/3分钟）
- `--accounts`: 账号数量（批量生成差异化内容）
- `--batch`: 批量模式（为多个账号生成内容）
- `--variants`: A/B测试变体数量
- `--market`: 目标市场（美区/欧盟/日本/东南亚）
- `--platform`: 发布平台（TikTok/抖音/Instagram/Bilibili）

## A/B测试框架

### 测试维度

**维度1：开头钩子**
- A型：痛点型（"厨房炸了？"）
- B型：疑问型（"你知道..."）
- C型：直接型（"这个神器..."）

**维度2：视频时长**
- A型：15秒（单点）
- B型：30秒（核心）
- C型：60秒（完整）

**维度3：内容风格**
- A型：幽默搞笑
- B型：专业认真
- C型：真实分享
- D型：故事情感

**维度4：BGM配合**
- A型：卡点音乐
- B型：无音乐
- C型：地方音乐

## 内容质量控制

### 必检项
- ✅ 开头3秒有冲击力
- ✅ 核心卖点在10秒内展示
- ✅ CTA清晰（购物车/关注）
- ✅ Hashtags合理数量（5-15个）
- ✅ BGM适合内容类型
- ✅ 时长符合平台偏好

### 避免项
- ❌ 开头3秒模糊不清
- ❌ 核心卖点过于分散
- ❌ CTA不清晰或过晚
- ❌ Hashtags过多（>20个）
- ❌ BGM不匹配节奏
- ❌ 时长过长（>45秒）
