# TikTok内容分析技能

深度分析TikTok热门视频内容，提取成功要素和创作策略。

## 信息

- **Video**: 1.0.0
- **Author**: OpenClaw Team
- **Tags**: tiktok, content-analysis, video, viral
- **Icon** 📺

## 功能

- 📊 分析热门视频数据（播放量、点赞、评论、分享）
- 🎬 识别爆款视频的共同特征
- 💡 提取爆款要素（开头3秒、音乐、挑战、字幕）
- 🔧 生成内容优化建议
- 📈 关联话题发现（基于Hashtag和评论区）

## 使用方法

```bash
# 分析特定视频
openclaw run tiktok-content-analyzer --url="video_url" --deep

# 批量分析Top N视频
openclaw run tiktok-content-analyzer --topic="厨房好物" --top=20

# 分析特定Hashtag下的热门视频
openclaw run tiktok-content-analyzer --hashtag="#好物推荐" --top=10
```

## 参数说明

- `--url`: TikTok视频链接
- `--topic`: 搜索关键词
- `--hashtag`: 特定Hashtag
- `--top`: 分析Top N个
- `--deep`: 深度分析（包含评论区分析）
- `--output`: 输出文件路径

## TikTok爆款特征分析

### 视频时长
- 15秒：单一卖点，快速转化
- 30秒：核心展示 + CTA
- 60秒：深度内容 + 教学
- 3分钟：完整教程或故事

### 开头钩子类型（前3秒）

**视觉冲击型**：
- 产品使用演示
- 对比展示（使用前后）
- 美食特写/特写
- 细节特写（质感放大）

**痛点打击型**：
- 直接提出问题
- 展示痛点（脏乱厨房、找不到东西）
- 情感共鸣（家庭聚餐、节日聚会）

**悬念型**：
- "你知道..."
- "这招你学会了吗"
- "最后3个方法..."

**直接展示型**：
- 产品快速演示
- 效果即时可见
- "5分钟搞定XXX"

### 内容结构

**AIDA模型（新加坡营销常用）**：
- Attention（吸引注意）：0-3秒
- Interest（激发兴趣）："别让XXX再..."
- Desire（创造欲望）：痛点+解决方案
- Action（行动号召）：点击购物/链接

**三段式结构**：
- 开头：痛点钩子 + 产品展示
- 中段：功能演示 + 使用方法
- 结尾：CTA + 总结 + Hashtags

### 音乐选择策略

**流行趋势****：
- 跟随抖音热门榜单
- 使用卡点音乐（提升推荐）
- 地区化音乐：泰语歌、越南语歌

**音乐类型**：
- 轻快活泼
- 节奏感强
- 视频类型匹配

## 账面评价策略

### 避免的负面评价
- 质量问题（质量差）
- 虚假宣传（与实物不符）
- 价格争议（价格过高）

### 鼓励的正面评价
- 真实使用体验
- 清晰的产品展示
- 耐心热情

## 地区化建议

### 泰国市场
- 泰语文案优先

- 注重视觉风格

### 越南市场
- 越南音乐为主

### 马来西亚
