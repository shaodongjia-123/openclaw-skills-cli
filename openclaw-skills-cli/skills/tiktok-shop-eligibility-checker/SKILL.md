智能检查账号是否符合TikTok Shop开店要求，提供详细的分析和建议。

## 功能

- 🆔 开店资格全面评估
- 🌍 支持多市场检查（美区/欧盟/日本）
- 📋 材料清单自动生成

## 使用方法

```bash
openclaw run tiktok-shop-eligibility-checker --market="美区" --category="服装" --investment="5000"

openclaw run tiktok-shop-eligibility-checker --market="欧盟" --category="3C电子产品" --investment="20000" --full
```

## 参数说明

- `--market`: 目标市场（美区/欧盟/日本/东南亚）
- `--category`: 产品品类（服装/3C/家居/美妆等）
- `--investment`: 投资资金规模（$5000、$10000等）
- `--full`: 完整详细报告模式

## 市场准入差异

### 美区 🇺🇸
- **门槛**：中等，需美元结算账户
- **税务**：需要SS4表格，本地注册要求
- **物流**：建议使用海外仓（3-7天配送）
- **投资建议**：$5000起，首月$1000-$3000

### 欧盟 🇪🇺
- **门槛**：较高，多语言要求
- **税务**：VAT注册，可能需要多国VAT
- **物流**：需考虑EU内部配送（1-3天）
- **投资建议**：$10000起，首月$2000-$5000

### 日本 🇯🇵
- **门槛**：高，需本地实体
- **税务**：法人税+消费税
- **物流**：时效要求高（1-2天）
- **投资建议**：$20000起，首月$5000-$10000

### 东南亚 🌏
- **门槛**：相对较低
- **税务**：各国不同，需要当地合作伙伴
- **物流**：建议本地仓+跨境结合
- **投资建议**：$3000起，首月$1000-$2000

## 品类准入建议

### ✅ 推荐品类（成功率高）
- 服装/配饰（美妆/时尚）
- 家居用品（收纳/收纳）
- 3C小家电（电子产品）
- 食品饮料（保健食品）
- 母婴用品

### ⚠️ 谨慎品类（审核严格）
- 医疗器械
- 保健品（需资质）
- 玩具（安全标准）
- 钟表珠宝
- 大件家具

### ❌ 禁止品类（绝对不能做）
- 军事用品
- 毒品
- 非法物品
- 仿品
- 虚假产品

## 风险预警信号

### 🔴 高风险（直接失败）
- 资金不足（<$2000）
- 产品类目禁止
- 无营业执照
- 税务信息缺失
- 品牌侵权

### 🟡 中风险（需要优化）
- 资金紧张（$2000-$5000）
- 产品缺少资质证明
- 没有本地化运营计划
- 物流时效不确定
- 竞品竞争激烈

### 🟢 低风险（基本通过）
- 资金充足（>$5000）
- 产品类目开放
- 资料齐全
- 有运营经验

## 常见问题解答

**Q: 开店需要多久？**
A: 资料齐全通常3-7天，复杂情况可能延长至2-3周。

**Q: 可以一个人做吗？**
A: 第几个月建议2-3人团队（运营+客服+物流），规模化后需要5-10人。

**Q: 一定要有自己的品牌吗？**
A: 不一定要自有品牌，但需要品牌授权或者产品有质量保证。

**Q: 第一次开店应该选哪个市场？**
A: 建议：东南亚最低门槛，美区/欧盟市场最大但要求高。

**Q: 开店失败怎么办？**
A: 查清失败原因（资料/资质/产品），改进后可以重新申请。
