# 内容研究写作技能

自动收集资料并生成文章，支持多源数据。

## 信息

- **Version**: 1.0.0
- **Author**: OpenClaw Team
- **Tags**: research, writing, content
- **Icon**: 📝

## 使用方法

```bash
openclaw run content-research --topic="AI编程工具对比"
```

## 特性

- ✅ 多源数据收集
- ✅ 自动结构化内容
- ✅ Markdown格式输出
- ✅ 智能总结

## 示例

```bash
# 技术对比文章
openclaw run content-research --topic="AI编程工具对比"

# 趋势分析
openclaw run content-research --topic="前端框架2025趋势"

# 最佳实践
openclaw run content-research --topic="Web开发最佳实践"
```

## 输出格式

结构化的Markdown文章，包含：
- 概述
- 主要发现
- 建议
- 结论

## 数据源

- 搜索引擎
- 文档网站
- 技术博客
- 论坛讨论
- 官方文档
