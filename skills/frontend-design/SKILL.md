# 前端设计技能

自动生成美观的Tailwind CSS界面，拒绝蓝紫渐变丑UI。

## 信息

- **Version**: 1.0.0
- **Author**: OpenClaw Team
- **Tags**: frontend, ui, react, tailwind
- **Icon**: 🎨

## 使用方法

```bash
openclaw run frontend-design --prompt="创建一个现代化的电商网站首页"
```

## 特性

- ✅ 现代化设计风格
- ✅ 完整的响应式布局
- ✅ 可访问性友好
- ✅ 直接可用代码

## 示例

```bash
# 示例 1: SaaS产品着陆页
openclaw run frontend-design --prompt="创建一个SaaS产品着陆页"

# 示例 2: 个人博客
openclaw run frontend-design --prompt="设计一个技术博客页面"

# 示例 3: 登录表单
openclaw run frontend-design --prompt="制作一个现代化的登录注册表单"
```

## 输出格式

完整的HTML + Tailwind CSS代码，可以直接复制粘贴使用。

## 设计原则

1. **拒绝滥用渐变**：不使用蓝紫渐变等过时设计
2. **现代色彩系统**：使用slate/indigo/emerald等现代色系
3. **优秀的可读性**：确保文字对比度符合WCAG标准
4. **响应式优先**：移动端优先的设计理念
5. **无障碍友好**：包含适当的ARIA标签
