### 六、定价模板（智能定价）

#### 6.1 定价模板概念

**作用：**
- 自动计算售价（考虑采购价、物流费、平台费、利润率）
- 批量应用（一键应用到多款商品）
- 动态调整（修改模板后所有商品价格自动更新）

**定价公式：**
```
售价 = （采购成本 + 物流成本 + 平台佣金 + 利润）× 汇率
```

#### 6.2 Shopee定价模板

**定价因子：**
- 采购价
- 物流费首重/续重
- 平台佣金费（5%-8%）
- 服务费（3%）
- 利润率（如30%）

**定价模板创建步骤：**
```
进入「定价模板」→「Shopee定价模板」
                      ↓
点击「创建模板」
                      ↓
填写以下字段：
  - 模板名称（如：Shopee泰国家居模板）
  - 默认货币（泰铢）
  - 平台佣金（5.5%）
  - 服务费（固定3%）
  - 物流费模板（选择泰国有物流商）
  - 利润率（如30%）
  - 其他费用（如打包费：฿5）
                      ↓
保存模板
```

**引用模板方法：**
```
发布商品时 → 选择「引用定价模板」
              ↓
选择模板（如：Shopee泰国家居模板）
              ↓
输入采购价或成本价
              ↓
系统自动计算售价
```

#### 6.3 TikTok定价模板

**TikTok跨境店定价配置：**

```
定价因子：
  - 采购成本
  - 物流费（泰国本土物流）
  - 平台佣金（5%）
  - 支付手续费（1.5%）
  - 利润率（如35%）
```

**TikTok全球店定价模板：**
```
跨境店支持引用模板（自动应用汇率换算）
```

**定价模板售价试算：**
```
进入「定价模板」→「定价模板售价试算」
                      ↓
输入采购价
                      ↓
查看售价结果
```

### 七、订单处理（第六步）

妙手ERP支持3种订单处理方式：

#### 7.1 方式1：代发采购（最常用）

**适用场景：** 无库存，线上拿货

**流程：**
```
Shopee订单同步到「订单处理」
         ↓
选择订单 → 点击「代发采购」
         ↓
关联货源平台（如1688）
         ↓
自动生成采购单 → 提交采购
         ↓
采购成功 → 生成面单 → 打印面单
         ↓
等待国际物流配送至买家
```

**代发采购 vs. 直接采购：**
- **代发采购**：直接链接1688/淘宝下单
- **直接采购**：采购模块手动创建采购单，走自己的库存

**支持的代发采购平台：**
- 1688跨境分销
- 淘宝跨境（Tao World）
- Alibaba国际站货盘
- Doba美区货盘
- 51mbk、CJDropshipping等

#### 7.2 方式2：调用库存发货（自营仓）

**适用场景：** 有自有仓库，部分备货

**流程：**
```
订单同步 → 选择订单 → 点击「调用库存」
         ↓
系统检查自营仓库存
         ↓
有库存 → 生成发货单 → 打印面单 → 拣货出库
无库存 → 提示补货或调用三方仓
```

**自营仓设置：**
```
进入「仓库模块」→「自营仓仓库清单」
           ↓
添加仓库（如：曼谷自营仓）
           ↓
设置货架位、商品SKU
           ↓
同步库存到店铺产品
```

**自营仓发货操作：**
```
订单处理 → 选择订单 → 「调用库存发货」
           ↓
系统自动生成拣货单
           ↓
按拣货单拣货 → 扫描入库
           ↓
生成面单 → 打印面单
           ↓
发物流
```

#### 7.3 方式3：调用三方仓（海外仓）

**适用场景：** 使用第三方海外仓或平台仓

**三方仓类型：**
- 第三方海外仓（100+家服务商）
- 平台仓（如Temu合作认证仓）
- FBA三方仓（亚马逊仓）

**授权三方仓（以泰国为例）：**
```
进入「仓库模块」→「第三方仓库授权」
                       ↓
选择泰国海外仓服务商
（如：海仓国际海外仓、易速菲海外仓2.0）
                       ↓
授权服务商账号
                       ↓
推送到海外仓的商品列表
```

**三方仓发货流程：**
```
订单同步 → 选择订单 → 点击「推送到三方仓」
         ↓
系统自动向海外仓创建出库指令
         ↓
海外仓拣货出库 → 发物流
         ↓
物流轨迹同步到妙手ERP
```
