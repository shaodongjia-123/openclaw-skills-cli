为 OpenClaw 应用添加飞书云文档读取权限，实现"一劳永逸"的文档访问能力。

## 配置步骤（3分钟搞定）

### 第1步：登录飞书开放平台

1. 打开浏览器，访问：https://open.feishu.cn/app
2. 使用飞书账号登录
3. 进入应用管理后台

### 第2步：找到 OpenClaw 应用

- 应用ID：`cli_a91a3482f2b89bb4`
- 应用名称：OpenClaw（或你自定义的名称）
- 点击进入应用详情页

### 第3步：添加云文档权限

**操作路径：**
```
左侧菜单 → 权限管理 → 添加权限
```

**必须添加的权限：**
- [x] `docx:document:readonly` - 读取云文档内容

**推荐添加的权限（可选）：**
- [ ] `docx:document:readonly:comment` - 读取文档评论
- [ ] `docx:document:readonly:revision` - 读取修订记录

**操作截图示意：**
```
[权限管理页面]
├─ 搜索框：输入 "docx"
├─ 结果列表：
│   ├─ ☐ docx:document:readonly
│   ├─ ☐ docx:document:readonly:comment
│   └─ ☐ docx:document:readonly:revision
└─ [确定] 按钮
```

### 第4步：保存并发布

1. 点击「确定」保存权限配置
2. 点击「发布」使配置生效
3. 等待飞书审核（通常即时生效）

---

## 第5步：重启 OpenClaw Gateway

权限添加后，必须重启 Gateway 才能加载新权限：

```powershell
# 停止现有 Gateway
taskkill /F /IM node.exe

# 等待 2 秒
Start-Sleep 2

# 重新启动 Gateway
node "C:\Users\MECHREVO\AppData\Roaming\npm\node_modules\openclaw\openclaw.mjs" gateway run
```

或使用快捷命令：
```bash
openclaw gateway restart
```

## 验证权限是否生效

重启 Gateway 后，在 OpenClaw 聊天窗口发送：

```

```

阿拓二号应返回包含以下内容的权限列表：

```json
{
  "granted": [
    {
      "name": "docx:document:readonly",
      "type": "tenant"
    }
  ]
}
```

## 常见问题

### Q1: 添加权限后还是读不了文档？
**A:** 确认已重启 Gateway，权限加载需要重新初始化。

### Q2: 提示"权限不足"？
**A:** 检查是否勾选了 `docx:document:readonly`，并确认已发布应用。

### Q3: 可以读取所有飞书文档吗？
**A:** 只能读取应用成员有权限访问的文档，受飞书权限体系管控。

### Q4: 读取文档有频率限制吗？
**A:** 飞书 API 有调用配额，具体查看飞书开放平台文档。
